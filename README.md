# Feedback Agent Sample Product

This is a tiny sample product repo for testing the Feedback Agent SDK against GitHub APIs.

## Run the Sample App

```bash
npm install
npm run dev
```

Open the Vite local URL, usually `http://localhost:5173`.

Click **Export CSV** to reproduce the intentional bug.

Push this folder to a GitHub repository, then configure:

```bash
GITHUB_OWNER=your_github_username_or_org
GITHUB_REPO=feedback-agent-sample-product
GITHUB_BRANCH=main
```
